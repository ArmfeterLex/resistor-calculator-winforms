# Resistor Calculator WinForms

Компонент на C# Windows Forms для расчета общего сопротивления цепи. Выбор последовательного или параллельного соединения, ввод значений резисторов через TrackBar, автоматический расчет.

## Функциональность

- Ввод сопротивлений R1 и R2 через TrackBar
- Выбор типа соединения: последовательное или параллельное
- Автоматический расчет при любом изменении
- Защита от деления на ноль
- Округление результата до двух знаков

## Интерфейс

- trackBar1 — выбор R1
- trackBar2 — выбор R2
- radioButtonSequential — последовательное соединение
- radioButtonParallel — параллельное соединение
- lblResistor1 — отображение R1
- lblResistor2 — отображение R2
- lblTotalResistance — отображение общего сопротивления

## Куда можно встроить

- В инженерные калькуляторы
- В образовательные приложения по физике и электротехнике
- В симуляторы электрических цепей
- В программы для расчёта электронных схем

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При параллельном соединении и одном из резисторов равном нулю, общее сопротивление принимается равным нулю (короткое замыкание). Результат округляется до двух знаков.
